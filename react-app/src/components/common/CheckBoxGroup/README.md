**属性**

1. chooseDatas：选中的元素
2. datas: 一个对象数组
    1. value 多选框的值
    2. text 多选框的文本

```js
[
    { value: 'football', text: '足球' },
    { value: 'basketball', text: '篮球' },
    { value: 'movie', text: '电影' },
    { value: 'music', text: '音乐' },
    { value: 'other', text: '其他' },
];
```

3. name： 每一个多选框的 name 属性
4. onChange: 状态改变时发生的事件
